# 🎮 Simon Says Game

A simple and interactive **Simon Says Game** built using **HTML, CSS, and JavaScript**.  
Test your memory by repeating the pattern of colors correctly!

---

## 🚀 Features

- 🎯 Random color pattern generation
- 🧠 Memory-based gameplay
- 🔊 Sound effects for each button
- ✨ Button animations (press & flash)
- ❌ Game Over detection
- 🔁 Restart game with key press
- 📱 Responsive design (works on desktop & mobile)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎮 How to Play

1. Press any key to start the game
2. Watch the pattern carefully
3. Click the buttons in the same order
4. Each level adds a new step to the pattern
5. If you press the wrong button → Game Over ❌
